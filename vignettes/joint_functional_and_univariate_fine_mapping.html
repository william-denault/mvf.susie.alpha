Joint functional and univariate fine mapping with multfSuSiE

library(mvf.susie.alpha)

multfSuSiE

Here we showcase the implementation of the a Bayesian variable selection method called the “multi Functional and univariate Sum of Single Effects” ({}) model that extends the {} and {} model of Denault et al.1 to jointly fine-map functional and univariate phenotypes. The methods implemented here are particularly well-suited to settings where some of the X variables are highly correlated, and the true effects are highly sparse (e.g. <20 non-zero effects in the vector b). One example of this is the joint genetic fine-mapping molecular measurements from sequencing assays (ChIP-seq, ATAC-seq, which are functional phenotypes ) and gene expression (univariate). However, the methods should also be useful more generally (e.g. detecting change in time series).

###Model Description When analysing a single functional phenotype with a single univariate phenotype, multfSuSiE fits the following linear regressions $ {Y_f} = {XB_f} + {E_f}$ and $ {Y_u} = {XB_u} + {E_u}$; where $ {Y_f}$ is a N x T matrix containing the observed curves, \(Y_u\) is N vector containning the univaraite phenotype, \(\ {X}\) is the N × p matrix of single nucleotide polymorphisms (SNP) in the region to be fine-mapped. \({B_f}\) is a p × T matrix whose element \(b_{jt}\) is the effect of SNP \(j\) on the trait at location t, \(B_u\) is a p vector whose element \(b_{j}\) is the effect of SNP \(j\) on the univariate trait, \({E_f}\) and \(E_u\) are noise terms considered to be normally-distributed with variance \(\sigma_f^2\) and \(\sigma_u^2\) respectively. Analogous to regular fine-mapping, the matrix \({B_f}\) and the vecotr \(B_u\) are expected to be sparse, i.e., most rows/entries will be zero. Non-zero rows/entries correspond to SNPs that do affect the trait. The goal is to identify the non-zero rows/entries of \({B_f}\) and \(B_u\). We extended {} and {} to functional traits by writing \({B_f}= \sum_l^L {B_f}^{(l)}\) and \({B_u}= \sum_l^L {B_u}^{(l)}\), where each “single-effect” doublet (matrix,vector) \({B_f}^{(l)},{B_u}^{(l)}\) has one non-zero entry 2, corresponding to a single causal SNP. We devised a fast variational inference procedure~ for fitting this model by first projecting the data in the wavelet space and then using an adaptation of the efficient iterative Bayesian stepwise selection (IBSS) approach proposed by Wang and colleagues .

The output of the fitting procedure is a number of “Credible Sets” (CSs), which are each designed to have high probability to contain a variable with non-zero effect, while at the same time being as small as possible. You can think of the CSs as being a set of “highly correlated” variables that are each associated with the response: you can be confident that one of the variables has a non-zero coefficient, but they are too correlated to be sure which one.
